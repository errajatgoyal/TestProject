## Introduction

This project is made to demonstrate the use of automation technology in order to automate the validation of 2 example scenarios on the Halo powered website.

## Prerequisites

- Intelli-J IDEA or a similar IDE
- Java installed in your machine (version 8 or higher)
- Knowledge and hands on Java, Selenium, Cucumber, Maven and TestNG

## Installation

Simply open the project in the intelli-J IDEA or a similar IDE once downloaded.
It should automatically install the dependencies.

## Running the Tests

The tests can be run via right clicking on the TestRunner class itself.
or, executing the command:-

```bash
mvn clean test
```

## Test Scenarios

1. Test Scenario 1: Checks the user landed on the Halo Powered home page should be able to see the correct mission statement visible on the homepage.
2. Test Scenario 2: Checks the user is able to submit the contact details successfully.
